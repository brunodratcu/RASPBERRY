AZURE CREDENTIALS EXTRACTION

PORTAL ACCESS
Open your web browser and navigate to the Azure Portal. Sign in with your Microsoft account credentials.
URL: https://portal.azure.com
DIRECTORY NAVIGATION
In the search bar at the top, type "entra id" and select "Microsoft Entra ID" from the results.
APPLICATION REGISTRY
In the left sidebar, locate and click on "App registrations".
NEW REGISTRATION
Click the "New registration" button to create a new application.
APPLICATION CONFIGURATION
Fill in the registration form with these exact values:
Name: Space Mirror - [Your Name] Supported account types: "Accounts in any organizational directory and personal Microsoft accounts" Redirect URI (optional): Platform: Web URL: http://localhost:5000/callback
CREDENTIAL EXTRACTION
After creating the app, you'll be on the Overview page. Copy these two values:
• Application (client) ID → This is your CLIENT ID • Directory (tenant) ID → This is your TENANT ID
SECRET GENERATION
In the left sidebar, click "Certificates & secrets", then click "New client secret". Add a description and set expiration, then click "Add".
⚠️ CRITICAL: Copy the "Value" field immediately! This secret will never be shown again.
PERMISSION CONFIGURATION
Click "API permissions" in the left sidebar, then "Add a permission". Select "Microsoft Graph" → "Delegated permissions".
Add these permissions: • Calendars.Read • User.Read Then click "Grant admin consent for [Directory]"
DEVICE PREPARATION
Ensure your Pico 2W is running the Magic Mirror client firmware and connected to WiFi.
Required: Pico 2W with MQTT client code WiFi connection with internet access
TOPIC CONFIGURATION
The Pico 2W should connect to the MQTT broker and subscribe to the registration topic.
MQTT Broker: test.mosquitto.org:1883 Registration Topic: [Loading...]
DEVICE REGISTRATION
The device will automatically appear in the scan results when it connects to MQTT.
APPROVAL PROCESS
Once detected, approve the device to start receiving calendar events.
