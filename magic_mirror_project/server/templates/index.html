SINCRONIZAÇÃO DE DISPOSITIVOS MQTT

Monitore e gerencie dispositivos Pico 2W conectados para sincronização de calendário

Clique em "PROCURAR DISPOSITIVOS" para buscar dispositivos Pico 2W

PROTOCOLO DE CONEXÃO PICO 2W

CONFIGURAR APLICAÇÃO
Configure sua aplicação com estes parâmetros:
Nome: Space Mirror - [Seu Nome] Tipos de conta: Contas em qualquer diretório organizacional e contas pessoais da Microsoft URI de Redirecionamento: http://localhost:5000/callback
EXTRAIR IDENTIFICADORES
Na página de Visão Geral, copie estes valores críticos:
• ID da Aplicação (cliente) → ID DO CLIENTE • ID do Diretório (locatário) → ID DO LOCATÁRIO
GERAR CHAVE SECRETA
Navegue para "Certificados e segredos" → "Segredos do cliente" → "Novo segredo do cliente"
Copie o VALOR SECRETO imediatamente - ele só aparece uma vez!
CONFIGURAR PERMISSÕES
Vá para "Permissões da API" → "Adicionar uma permissão" → "Microsoft Graph" → "Permissões delegadas"
Permissões necessárias: • Calendars.Read • User.Read Em seguida, clique em "Conceder consentimento do administrador"

EXTRAÇÃO DE CREDENCIAIS AZURE

ACESSO AO PORTAL
Abra seu navegador web e navegue até o Portal Azure. Faça login com suas credenciais da conta Microsoft.
URL: https://portal.azure.com
NAVEGAÇÃO NO DIRETÓRIO
Na barra de pesquisa no topo, digite "entra id" e selecione "Microsoft Entra ID" dos resultados.
REGISTRO DE APLICAÇÃO
Na barra lateral esquerda, localize e clique em "Registros de aplicativo".
NOVO REGISTRO
Clique no botão "Novo registro" para criar uma nova aplicação.
CONFIGURAÇÃO DA APLICAÇÃO
Preencha o formulário de registro com estes valores exatos:
Nome: Space Mirror - [Seu Nome] Tipos de conta com suporte: "Contas em qualquer diretório organizacional e contas pessoais da Microsoft" URI de Redirecionamento (opcional): Plataforma: Web URL: http://localhost:5000/callback
EXTRAÇÃO DE CREDENCIAIS
Após criar o aplicativo, você estará na página de Visão Geral. Copie estes dois valores:
• ID da Aplicação (cliente) → Este é seu ID DO CLIENTE • ID do Diretório (locatário) → Este é seu ID DO LOCATÁRIO
GERAÇÃO DE SEGREDO
Na barra lateral esquerda, clique em "Certificados e segredos", depois clique em "Novo segredo do cliente". Adicione uma descrição e defina a expiração, depois clique em "Adicionar".
⚠️ CRÍTICO: Copie o campo "Valor" imediatamente! Este segredo nunca será mostrado novamente.
CONFIGURAÇÃO DE PERMISSÕES
Clique em "Permissões da API" na barra lateral esquerda, depois "Adicionar uma permissão". Selecione "Microsoft Graph" → "Permissões delegadas".
Adicione estas permissões: • Calendars.Read • User.Read Depois clique em "Conceder consentimento do administrador para [Diretório]"
PREPARAÇÃO DO DISPOSITIVO
Certifique-se de que seu Pico 2W está executando o firmware cliente Magic Mirror e conectado ao WiFi.
Necessário: Pico 2W com código cliente MQTT Conexão WiFi com acesso à internet
CONFIGURAÇÃO DE TÓPICOS
O Pico 2W deve se conectar ao broker MQTT e se inscrever no tópico de registro.
Broker MQTT: test.mosquitto.org:1883 Tópico de Registro: [Carregando...]
REGISTRO DO DISPOSITIVO
O dispositivo aparecerá automaticamente nos resultados da busca quando se conectar ao MQTT.
PROCESSO DE APROVAÇÃO
Uma vez detectado, aprove o dispositivo para começar a receber eventos do calendário.
