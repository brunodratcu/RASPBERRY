⚡ PERMISSÕES DE API - PASSO CRÍTICO

Configure as permissões EXATAS necessárias para o funcionamento completo do sistema. Todas as 5 permissões são OBRIGATÓRIAS!

📋 PASSO A PASSO DETALHADO:
1️⃣ Clique em "API permissions" (menu lateral esquerdo)
2️⃣ Clique em "+ Add a permission"
3️⃣ Selecione "Microsoft Graph"
4️⃣ Selecione "Delegated permissions" (NÃO Application)
5️⃣ Marque TODAS as 5 permissões abaixo ⬇️
✅ MARQUE ESTAS 5 PERMISSÕES:
✅ openid
Categoria: OpenId
Função: Identificação básica do usuário durante login OAuth
Obrigatória: SIM - Para autenticação funcionar
✅ profile
Categoria: OpenId
Função: Obter nome completo e foto do perfil
Obrigatória: SIM - Para mostrar "Comandante: [Nome]"
✅ email
Categoria: OpenId
Função: Obter endereço de email do usuário
Obrigatória: SIM - Para identificação única
✅ offline_access
Categoria: OpenId
Função: Gera refresh token para renovação automática
Obrigatória: SIM - Evita fazer login toda hora
✅ Calendars.Read
Categoria: Calendars (precisa buscar manualmente)
Função: Ler eventos do calendário do Outlook
Obrigatória: SIM - É o objetivo principal do sistema!
🔍 COMO ENCONTRAR CADA PERMISSÃO NO AZURE: 1. API permissions → + Add a permission 2. Microsoft Graph → Delegated permissions 3. Use a BUSCA para encontrar cada uma: Na caixa "Select permissions", digite: 📌 "openid" → Marque ✅ 📌 "profile" → Marque ✅ 📌 "email" → Marque ✅ 📌 "offline" → Aparece "offline_access" → Marque ✅ 📌 "calendars" → Aparece "Calendars.Read" → Marque ✅ 4. Clique em "Add permissions" (botão azul no final) 5. ⚠️ IMPORTANTE: NÃO clique em "Grant admin consent" (não é necessário para contas pessoais!) ✅ Resultado esperado: Você verá as 5 permissões listadas com status "Not granted" Isso é NORMAL para contas pessoais! O consent será dado quando o usuário fizer login.
⚠️ ERROS COMUNS:
❌ Selecionar "Application permissions" em vez de "Delegated"
❌ Esquecer de marcar "offline_access" (sem ela, token expira rápido)
❌ Não buscar "Calendars.Read" corretamente
❌ Tentar clicar em "Grant admin consent" (ignora esse botão!)
